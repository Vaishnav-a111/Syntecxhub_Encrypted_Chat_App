# Encrypted Chat App

A secure encrypted chat application developed using Python socket programming and AES encryption.

## Features

- AES encrypted messaging
- TCP socket communication
- Multiple client support
- Message logging
- Threading/concurrency

## Technologies Used

- Python
- Socket Programming
- AES Encryption
- Threading
- PyCryptodome

## Installation

Install required library:

pip install pycryptodome

## Run Server

python server.py

## Run Client

python client.py

## Project Structure

server.py -> Server-side code  
client.py -> Client-side code  
chat_logs.txt -> Stores chat history  
requirements.txt -> Required libraries  
README.md -> Project documentation
